Steps:
1.Click the color box to change color
2.Choose "Geometry" or "Curve"
   Geometry - Set arguments
   Curve - Choose what curve you want to draw
3.Draw by clicking on the canvas above
   (A curve can be adjusted by dragging the red points
   around it)
4.Adjust your drawing by the Function "Transform"
   Move - Drag your drawing on the canvas
   Rotate / Zoom - Set arguments and click "+" / "-"
   Reflect - Set arguments and click "X" / "Y"
5.Click "3D model" to generate a 3D model
6.Download your drawing after generating 3D model
